# DSCI 372 Introduction to Machine Learning

Coursework, homework, and projects for DSCI 372. Covers foundational ML concepts using Python — from supervised learning to model evaluation and beyond.

---

## Course objectives
- Understand core concepts and methods in machine learning
- Apply ML techniques to solve a variety of real-world problems
- Build a foundation for further study in any ML-related area

---

## Topics covered
| Topic | Methods |
|---|---|
| Supervised learning | linear/logistic regression, classification |
| Tree-based methods | decision trees, random forests, boosting |
| Model evaluation | cross-validation, bias-variance tradeoff |
| Unsupervised learning | k-means clustering, PCA |
| Neural networks | perceptrons, backprop, intro to deep learning |
| Feature engineering | preprocessing, scaling, pipelines |

---

## Stack
`Python 3.x` `scikit-learn` `pandas` `numpy` `matplotlib` `seaborn`

> All code written in Python as required by the course. Notebooks are best viewed in Jupyter.
